Currency Survey Analysis

This project analyzes survey data related to currency usage, preferences, and awareness. It focuses on extracting meaningful insights from raw responses using data cleaning and basic analytics.

##What this project does

Collects and processes survey responses on currency usage
Cleans inconsistent or missing data
Performs exploratory data analysis
Identifies trends and user preferences
Presents results in a clear, structured format

##Key features

Data preprocessing and validation
Summary statistics and trend analysis
Simple visualizations for insights
Structured output for easy interpretation

##Tech used

Python
Pandas
NumPy
Matplotlib or Seaborn

##Use case

You use this project to understand how people interact with currency systems. It helps in decision making and user behavior analysis.
